## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-file`
- `allow-copy-file`
- `allow-move-file`
- `allow-remove-file`
- `allow-create-folder`
- `allow-create-file`
- `allow-list-files`
- `allow-list-bookmarks`
- `allow-create-bookmark`
- `allow-remove-bookmark`
- `allow-list-tags`
- `allow-create-tag`
- `allow-create-tag-bind`
- `allow-remove-tag-bind`
